# 메이플 대시보드 — 개인정보처리방침

_최종 수정: 2026-06-29_

## 요약
**메이플 대시보드**는 모든 데이터를 **사용자의 브라우저 안(`chrome.storage.local` / `localStorage`)에만** 저장합니다. 개발자 서버가 없으며, 어떤 데이터도 개발자나 제3자에게 전송·판매·공유하지 않습니다. 광고·추적·분석 도구를 사용하지 않습니다. 외부 통신은 아래에 명시된, 기능 수행에 직접 필요한 서비스 호출에 한합니다.

## 다루는 데이터와 용도
| 데이터 | 출처/저장 | 용도 | 외부 전송 |
|---|---|---|---|
| 넥슨 Open API 키 | 사용자 입력 → 로컬 저장 | 넥슨 API 인증 | 넥슨 API 호출 시에만 헤더로 전송. 그 외 전송 없음 |
| 캐릭터·스케줄러·유니온·공지·이벤트·캐시샵 정보 | `open.api.nexon.com` 조회 → 로컬 캐시 | 캐릭터 카드·주간 보스 현황·보스 수익·공지 표시 | 없음(조회만) |
| 테스트 서버 공지 | `maplestory.nexon.com` 공개 페이지 조회 | 테스트 서버 섹션 표시 | 없음(조회만) |
| 위치(위도/경도) | 브라우저 위치 권한(런타임 동의) | 날씨 표시 | `api.open-meteo.com`(Open-Meteo)로 좌표 전송. 좌표 자체는 영구 저장하지 않음(날씨 결과만 캐시) |
| 북마크 | `chrome.bookmarks` 조회·관리 | 새 탭에 북마크 표시 및 사용자의 추가·수정·삭제·폴더 정리 | 없음(로컬 처리만) |
| 자주 가는 사이트 | `chrome.topSites` 조회 | '자주 가는 사이트' 위젯 | 없음(로컬 표시만) |
| 테마·배경 이미지·각종 표시 설정 | 사용자 입력 → 로컬 저장 | 개인화 | 없음 |

## 저장 위치
전적으로 사용자 기기 내부(`chrome.storage.local`, 테마는 `localStorage`). 개발자 측 서버·데이터베이스 없음.

## 데이터 삭제
- 확장 설정의 **🗑 전체 초기화**(북마크 배치 제외 모든 데이터·캐시 삭제), 또는
- 크롬에서 확장 프로그램 제거 시 로컬 데이터가 함께 삭제됩니다.

## 제3자 서비스
기능 수행을 위해 사용자가 켠 기능에 한해 다음 공개 서비스에 요청을 보냅니다: 넥슨 Open API, 메이플스토리 공식 웹(테스트 서버 공지), Open-Meteo(날씨). 각 서비스의 개인정보 처리에 대해서는 해당 서비스 정책을 따릅니다.
